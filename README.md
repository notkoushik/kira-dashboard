# KIRA — Koushik's Intelligent Recruitment Agent

A production job search dashboard built as a single-file SPA with real-time Supabase backend, Gmail integration, Sentry error monitoring, and Chart.js analytics.

## Features

- **Command Center** — KPI dashboard with live clock, progress ring, daily work logs
- **Daily Planner** — 4-task daily routine with streak tracking
- **Jobs Tracker** — Full CRUD with search, filter, sort, bulk actions
- **Apply Queue** — AI-scored job queue with one-click apply
- **HR Contact Manager** — Email automation with draft generation
- **Inbox Monitor** — Gmail integration for recruitment emails
- **Analytics** — 6 Chart.js visualizations with activity heatmap
- **Settings** — Quick Paste parser, CSV/JSON export, system status

## Tech Stack

- Pure HTML/CSS/JS (ES6+ modules)
- Supabase (PostgreSQL backend)
- Chart.js 4.x
- Lucide Icons
- Sentry (error monitoring)
- Inter font (Google Fonts)

## Live

Deployed on Vercel.
